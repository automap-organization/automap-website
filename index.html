Automap Desktop

Automap Desktop est un outil portable vous permettant de créer des cartes en format html sur votre appareil Windows.

Automap Web

Automap Web vous permet de créer des cartes en format kml sur notre site, sans rien télécharger, et accessible sur Windows, Linux et Mac OS.

Automap Python

Automap vous propose de générer des cartes, avec un module simple d'utilisation en Python et accessible à tous les développeurs Python.
